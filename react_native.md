# React Native Quick Reference

## Styles
```js
/* NO PADDING */
noPadding: {
    padding: 0,
    paddingLeft: 0,
    paddingRight: 0,
    paddingTop: 0,
    paddingBottom: 0
}

/* NO MARGIN*/
noMargin: {
    margin: 0,
    marginLeft: 0,
    marginRight: 0,
    marginTop: 0,
    marginBottom: 0
}
```
